Looking into the Popularity of Animal Crossing: New Horizons

Stephanie Grasso

What is this?

Final Project for Data Visualization that will look into the rise of popularity in the Nintendo game title Animal Crossing New Horizons. Specifically the traction it's gained throughout the years, the impact of sales, and how the audience responds to the many different animals the game has to offer.

> Alpha Report

Taking data from Google search trends we can see the volume of people's search of "Animal Crossing" throughout time. From the visualization we can see how the search activity spikes around the same time a new game is launched. This is especially true in the spike early in 2020.
